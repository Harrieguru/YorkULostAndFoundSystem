# Using GIT
Git is used to allow a group of programmers to work together.
Git works by making 'branches' and merging them.

EXAMPLE
Each group member works on specific parts of the program.
    ie 'branch/backend/login-authentication' by member 1 
    ie 'branch/frontend/login-page' by member 2
  
Then the branches are merged for the login usecase to work


# ROADMAP

# Starting (DO THIS ONCE)
1. Open vscode terminal
2. CD to folder you want store git project in
3. 'git clone https://github.com/your-team/project-name.git'
4. then cd to the project that is now in your folder

# Find latest main of project
5. git checkout main
6. git pull origin main


# Then, make a branch (your workspace) 
7. git checkout -b branch/feature/your-feature-name        (name of branch) ie. 'branch/feature/login'


# Finish your part of the code. Then,
8. git add . (stay at root folder of project)
9. git status
10. git commit -m "My part of project"
11. git status
12. git push origin branch/feature/your-feature-name


# Merging after branch is finished
13. go to github repository
14. find your branch and click the (...) and do a pull request
15. pull request will merge your branch with main


# Miscellaneous
16. run git reset if you accidentally git add . the node_module files
